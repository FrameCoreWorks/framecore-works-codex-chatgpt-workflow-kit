# Architecture

FrameCore Works Skill Kit is a provider-neutral workflow layer for Codex.

The system has four layers:

- Pipeline core: role routing, gates, handoffs, project state, and artifact templates.
- Production skills: brief, reference, research, direction, copy, prompts, QA, delivery, Humanizer, and HyperFrames knowledge.
- Expansion layer: Hipson Adapter with optional full Hipson connection.
- Governance layer: explicit workflow self-improvement logs and change proposals.

Agent source uses role IDs. Local display names are generated by onboarding and are not part of the public repo.
