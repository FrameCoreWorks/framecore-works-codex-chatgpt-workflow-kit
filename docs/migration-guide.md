# Migration Guide

Use this guide to migrate a private workflow into a public FrameCore Works Skill Kit-compatible setup.

1. Remove private project context.
2. Replace private names with role IDs.
3. Remove local absolute paths.
4. Remove private cloud folder IDs and delivery targets.
5. Remove provider-specific clients, keys, endpoint catalogs, and activation phrases.
6. Keep only reusable workflow logic, templates, gates, and handoffs.
7. Run privacy audit and validation.
