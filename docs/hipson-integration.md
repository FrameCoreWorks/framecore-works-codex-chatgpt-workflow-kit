# Hipson Integration

This repo includes a lightweight Hipson Adapter.

Current adapter scope:

- research maps
- internet mapping packets
- bounded instruction packets for pipeline agents
- review packets
- execution packets
- handoff support inside this workflow architecture

This is not the full Hipson system.

Full Hipson is maintained separately:

https://github.com/Hipson47/Hipson.git

Users can later connect the full Hipson repository to expand the workflow with repo scanning, delta scans, sidecar review agents, full Hipson knowledge packs, cross-repo orchestration, CLI commands, and complete Hipson workflow assets.
