# Hipson Adapter

This is the lightweight adapter, not the full Hipson system.

Current scope:

- research maps
- internet mapping packets
- bounded instruction packets
- review packets
- execution packets
- handoff support inside this workflow architecture

Full Hipson can be connected separately from:

https://github.com/Hipson47/Hipson.git

Do not replace workflow orchestration, perform full research by default, run external tools, or treat adapter output as verified truth.
