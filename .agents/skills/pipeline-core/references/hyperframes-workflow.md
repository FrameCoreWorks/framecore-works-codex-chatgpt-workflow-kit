# HyperFrames Workflow

HyperFrames is a coded-video production path.

Required inputs:

- Motion Direction Contract
- Storyboard Contract or scene structure
- Copy Pack
- asset list
- render constraints

Expected output:

- HyperFrames Production Brief
- scene timing
- composition structure
- GSAP animation notes
- captions and overlays
- render QA checklist
- delivery manifest requirements
