# Humanizer Routing

Use `humanizer` when copy needs to sound natural, specific, and less generic.

Common routes:

- `copy-voice` for VO, social copy, dialogue, taglines, and CTA polish
- `delivery-documentation` for final user-facing summaries
- `brief-architect` when messy input needs clearer phrasing without changing facts

Humanizer must preserve factual claims, exact required copy, legal wording, and user-approved terminology.
